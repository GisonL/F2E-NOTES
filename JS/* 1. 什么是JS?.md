# JS

javascript是一种专为网页交互而设计的脚本语言，由下列三个不同的部分组成：

一、 ECMAscript，由ECMA-262定义，提供核心语言功能。

二、文档对象模型(DOM)，提供访问和操作网页内容的方法和接口。

三、浏览器对象模型(BOM)，提供与浏览器交互的方法和接口。



> JavaScript的这三个组成部分，在当前五个主要浏览器(IE、Firefox、Chrome、Safari和Opera)中得到了不同程度的支持。其中，所有浏览器对ECMAscript第三版的支持大体都不错，而对ECMAscript5的支持程度越来越高，但对DOM的支持彼此相差比较多。对已经正式纳入HTML5标准的BOM来说，尽管各浏览器都实现了某些众所周知的共同特性，但其他特性还是会因浏览器而异。
>
> 注：现在的浏览器已完成对ES5的完全支持，对于ES6已完成绝大部分支持。
